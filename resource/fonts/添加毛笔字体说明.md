# 添加毛笔手写体字体说明

## ⭐ 推荐：霞鹜文楷（已设为默认）

**霞鹜文楷 (LXGW WenKai)** 是一款优秀的开源中文手写字体，清新自然，非常适合视频字幕。

### 快速下载

**方式一：GitHub下载（推荐）**
1. 访问: https://github.com/lxgw/LxgwWenKai/releases
2. 下载最新版本的 `LXGWWenKai-Regular.ttf` (约12MB)
3. 将文件复制到 `resource/fonts/` 目录
4. 重启应用即可

**方式二：国内镜像（更快）**
1. 访问: https://gitee.com/lxgw/LxgwWenKai/releases
2. 下载 `LXGWWenKai-Regular.ttf`
3. 复制到 `resource/fonts/` 目录
4. 重启应用

### 字体预览

- 清新手写风格，笔画流畅自然
- 支持简体中文、繁体中文、日文假名
- 适合各种视频场景

## 如何添加字体

1. **下载字体文件**
   - 从上述链接下载 `.ttf` 或 `.ttc` 字体文件

2. **放入字体目录**
   - 将字体文件复制到: `resource/fonts/` 目录

3. **重启应用**
   - 重启 MoneyPrinterTurbo 应用
   - 字体会自动出现在字体选择列表中

## 已默认设置

如果您下载了 **霞鹜文楷 (LXGWWenKai-Regular.ttf)**，该字体已被设置为默认字体。
